# 互联网垃圾网站列表

这里是收集到的互联网上的垃圾网站列表。

## 使用
您可以这样利用它：
* Chrome：您可以将它导入到 Chrome 的 [Personal Blocklist](https://chrome.google.com/webstore/detail/nolijncfnkgaikbjbdaogikpmpbdcdef) 插件中，用以屏蔽 Google 搜索结果中的垃圾网站。
* Firefox：*我并不是一个 FF 用户，希望您能帮助我找到这类插件*
